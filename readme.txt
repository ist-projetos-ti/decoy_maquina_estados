Arquivos para a máquina de estados

*Controller
controller possui uma função igual no views.py
Para rodar no raspberry, rodar o arquivo controller.py

*Setups
Os setups possuem a função setup para cada máquina de estado, pode utilizá-los para testar, basta trocar o import na função controller. Também pode utilizá-los como base para alterar a função setup no arquivo views.py.
Na versão final, haverá apenas um setup. É preciso separá-los por enquanto por causa da limitação de inputs e outputs.
